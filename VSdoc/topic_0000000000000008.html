Parameters

connectionString

Type: string

The connection string that this procedure requires. Procedures annotated with *Always Encrypted* will append the connection with "Column Encryption Setting=enabled"

procedureName

Type: string

Name of the procedure in the format [specificSchema].[PROCEDURE_NAME].

isJsonResult

Type: bool

Stored procedure annotation *Returns Json*. Informs the execution methods that the procedure returns Json.

isSingleResult

Type: bool

Stored procedure annotation *Single Result*. Informs the execution methods that the result is a single entity allowing Execute<T> vs. ExecuteList<T>.

isNonQuery

Type: bool

Stored procedure annotation *Non Query*. Informs the execution methods that the procedure does not return a value.

parameters

Type: Dictionary<string, Parameter>

Collection of parameters associated with the stored procedure.

parameterNameMap

Type: Dictionary<string, string>

A ParameterNameMap is a dictionary that maps a parameter name in the format @PARAMETER_NAME to any of the following variations FIRST_NAME, FirstName, and firstName. The mapping prevents case conversion on every set or get.
